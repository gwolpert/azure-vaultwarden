# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please do not open a public issue for security vulnerabilities.**

Instead, please report them via [GitHub Private Vulnerability Reporting](https://github.com/gwolpert/azure-vaultwarden/security/advisories/new).

### What to Include

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Any suggested fixes (if you have them)

### What to Expect

- We will acknowledge your report within 7 days
- We will provide an update on the status within 14 days
- We will work with you to understand and resolve the issue

## Scope

This security policy covers the Azure deployment templates and scripts in this repository. For security issues related to Vaultwarden itself, please report them to the [Vaultwarden project](https://github.com/dani-garcia/vaultwarden/security).
